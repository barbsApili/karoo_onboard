Python

Mac Instructions

  1. Check if Python is already installed by opening a terminal and type python3 --version or python --version to check if Python is already installed. If it is, you will see the version number. If not, you will see an error message.

  2. Install Homebrew (optional but recommended): Homebrew is a package manager for macOS that makes it easy to install and manage software packages. You can install it by running the following command in a terminal:
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

  3. Install Python using Homebrew: If you have Homebrew installed, you can install Python by running the following command in a terminal:
    brew install python

  4. Verify the installation: After the installation is complete, you can verify that Python was installed correctly by running the command below. You should see the version number of the Python interpreter:
    python3 --version or python --version in a terminal.

Linux Instructions

  1. Check if Python is already installed by opening a terminal and type which python3 to check if Python is already installed. If it is, you will see the version number. If not, you will see an error message.

  2. To install Python on a Linux machine, you can use various package managers provided by different distributions. Here's how you can do it for some popular distributions:
    • Ubuntu/Debian
      • sudo apt update
      • sudo apt install python3
    • Fedora
      • sudo dnf install python3
    • CentOS
      • sudo yum install python3
    • Arch Linux
      • sudo pacman -S python
    • OpenSUSE
      • sudo zypper install python3
  3. Verify the installation was successful using the same command in 1.

Docker

Mac Instructions

  1. Download Docker Desktop for Mac
    • Go to the Docker Hub website: Docker Hub
    • Click on the appropriate download for your computer
  2. Install Docker Desktop
    • Open the downloaded .dmg file
    • Drag the Docker icon to your Applications folder.
  3. Run Docker Desktop
    • Open Docker from your Applications folder.
    • You may be prompted to grant permission to install additional components. Follow the on-screen instructions to complete the installation.
  4. Start Docker Desktop
    • Open Docker from your Applications folder.
    • Docker Desktop will start and display a Docker icon in your menu bar.
  5. Verify Docker Installation
    • Open a terminal and run the following command to verify that Docker is installed and running using: docker --version
    • You should see the Docker version information printed to the terminal if Docker is installed correctly.
  6. Run Docker Containers (Optional)
    • You can now start using Docker to run containers. Try running a simple container to test the installation by running: docker run hello-world
    • Docker will download and run the "hello-world" container, and you should see a message indicating that Docker is working correctly.

Linux Instructions

To set up Docker on a Linux computer, you can follow these steps. The exact commands may vary slightly depending on your Linux distribution, but the overall process should be similar

  1. Update Package Index
    • Open your terminal and update the package index to ensure you have the latest information about available packages using: sudo apt update
  2. Install Required Packages
    • Install packages to allow apt to use a repository over HTTPS and to add Docker's official GPG key using:
      sudo apt install apt-transport-https ca-certificates curl software-properties-common
    • Drag the Docker icon to your Applications folder.
  3. Add Docker Repository
    • Add Docker's official GPG key using: curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo apt-key add -
    • Add the Docker repository to your system's list of sources using:
      sudo add-apt-repository "deb [arch=amd64] https://download.docker.com/linux/ubuntu $(lsb_release -cs) stable"
  4. Update Package Index Again
    • Update the package index once more to ensure you have access to Docker packages using: sudo apt update
  5. Install Docker CE (Community Edition):
    • Install the latest version of Docker CE using: sudo apt install docker-ce
  6. Verify Docker Installation
    • Check that Docker CE is installed correctly by running the following command, which should output the installed version of Docker using:
      docker --version
  7. Start Docker Service
    • Start the Docker service if it's not already running using: sudo systemctl start docker
    • Enable Docker to start on boot using: sudo systemctl enable docker

Kubernetes

Mac Instructions

  1. Assuming you have now installed Homebrew we can instal Kubernetes (K9s) immediately using: brew install k9s
  2. Install Python using Homebrew: If you have Homebrew installed, you can install Python by running the following command in a terminal:
    brew install python
  3. Verify Installation: After the installation is complete, you can verify that K9s is installed by running using:
    k9s version
  4. To run k9s simply just type in k9s in yout terminal.

Linux Instructions

To install K9s on a Linux machine, you can use the following steps. These steps assume you are using a Debian-based distribution (e.g., Ubuntu). If you are using a different distribution, the package manager commands may vary.

  1. Download the K9s Binary
    • Go to the K9s GitHub releases page: K9s Releases
    • Download the appropriate binary for your architecture. For Linux, you'll likely want the k9s_Linux_x86_64.tar.gz file, but make sure to check for the latest release and download the correct file.
  2. Extract the Binary
    • Open a terminal and navigate to the directory where you downloaded the tar.gz file.
    • Extract the contents of the file using the following command (replace k9s_VERSION.tar.gz with the actual name of the file you downloaded) using :tar -xzvf k9s_VERSION.tar.gz
  3. Move the Binary
    • Move the extracted binary to a directory in your PATH, such as /usr/local/bin, to make it executable system-wide using: sudo mv k9s /usr/local/bin/
  4. Make the Binary Executable
    • Make the binary executable by running using: sudo chmod +x /usr/local/bin/k9s
  5. Verify Installation
    • Verify that K9s is installed by running using: k9s version
  6. To run k9s simply just type in k9s in yout terminal.
